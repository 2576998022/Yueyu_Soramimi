# Text Analysis Request Format

请按以下格式提供您需要分析的文本：

## Input Requirements
- 提供完整的中文文本
- 保持原有的标点和空格
- 说明是歌词还是对话（如有需要）

## Expected Output
分析结果将包含：

1. 原文展示：
   [完整原文]

2. 分词结果：
   [用斜杠分隔的词组]

3. 分词说明：
   - 复合词标注
   - 空格位置
   - 特殊标点说明

## Example Request
"请分析以下歌词：我倾心 只等你回来时贴近"

## Example Response
原文展示：
我倾心 只等你回来时贴近

分词结果：
我/倾心/只/等/你/回来/时/贴近

分词说明：
- 复合词：倾心、回来
- 空格位置：位于"倾心"之后
- 特殊标点：无

## Additional Notes
- 如需特殊处理请说明
- 如有特定要求请提前告知
- 可以询问任何分词相关问题
