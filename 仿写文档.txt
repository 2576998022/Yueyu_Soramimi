# 粤语空耳格式仿写指南

## 一、基本格式结构
普通话原文
粤语空耳记音（汉字+拼音混合） 
简化记音（全拼音）

## 二、详细格式规则

1. 第一行（普通话）
- 使用完整的标准中文
- 保持原有标点符号
- 例：拦路雨偏似雪花

2. 第二行（粤语空耳）
- 规则1：保留能用汉字表达的音
  如：揽(lan)、楼(lou)、雨(yu)
- 规则2：使用拼音替代难以用汉字表达的音
  如：pin、gip、lei
- 规则3：特殊音标记方式
  - 长音：通常直接使用拼音
  - 入声字：常用 p、t、k 结尾
- 例：揽楼雨pin时雪花

3. 第三行（简化记音）
- 规则1：全部使用小写拼音
- 规则2：词与词之间用空格分隔
- 规则3：使用简化的拼音系统
- 例：lan lou yu pin ci xut fa

## 三、仿写步骤流程

1. 准备阶段
- 确定普通话原文
- 查找对应的粤语发音

2. 第一行书写
- 直接写入普通话原文
- 保持原有标点符号

3. 第二行转写
- 步骤1：标识可用汉字表达的音
- 步骤2：标识需要用拼音表达的音
- 步骤3：组合汉字和拼音

4. 第三行记音
- 将第二行完全转换为拼音
- 添加适当的空格分隔

## 四、示例分析

拦路雨偏似雪花            // 普通话原文
揽楼雨pin时雪花          // 粤语空耳（汉字+拼音混合）
lan lou yu pin ci xut fa  // 简化记音（全拼音）

分析：
- 汉字保留：揽(lan)、楼(lou)、雨(yu)、时(ci)、雪(xut)、花(fa)
- 拼音替代：pin（偏的粤语发音）
- 音节分隔：每个音节间保持空格

## 五、注意事项

1. 声调处理
- 简化记音通常省略声调标记
- 重点关注入声字的处理

2. 特殊音处理
- 粤语特有音需特别注意
- 入声字（p、t、k结尾）的准确标记

3. 格式一致性
- 保持三行格式的对齐
- 保持拼音书写的一致性

4. 常见错误避免
- 避免混用不同拼音系统
- 避免漏掉关键音节
- 保持空格的一致性
